# Critical Compass Shareable Beta

Upload the files from `github-pages/downloads/` as GitHub Release assets, or publish the `github-pages/` folder through GitHub Pages.

## Download Guide

- `critical-compass-plugin.zip`: Claude Cowork/custom plugin upload.
- `critical-compass-0.1.0.mcpb`: Claude Desktop Settings > Extensions.
- `critical-compass.skill` and `skill.zip`: fallback/reference mode.
- `checksums.txt`: SHA-256 checksums.

Claude's unverified developer/access warning is expected for local plugins/extensions.
