# Critical Compass Shareable Beta

Upload the files from `github-pages/downloads/` as GitHub Release assets, or publish the `github-pages/` folder through GitHub Pages.

## Download Guide

- `critical-compass-plugin.zip`: Claude Cowork, Claude Code, and Codex add-on upload.
- `critical-compass-0.1.0.mcpb`: Claude Desktop extension install through Settings > Extensions.
- `critical-compass.skill` and `skill.zip`: offline reference fallback.
- `checksums.txt`: SHA-256 checksums.

Claude's unverified developer/access warning is expected for locally shared add-ons and extensions.
